pk-spinner

CSS-only loading spinners. Use inline or as a block element.

Sizes

pk-spinner-sm
pk-spinner-md
pk-spinner-lg
pk-spinner-xl

Colors

primary
secondary
success
error
white

Inline (with text / inside button)

Loading data

Usage

<!-- Block (centered) -->
<span class="pk-spinner"></span>

<!-- Sizes -->
<span class="pk-spinner pk-spinner-sm"></span>
<span class="pk-spinner pk-spinner-lg"></span>
<span class="pk-spinner pk-spinner-xl"></span>

<!-- Colors -->
<span class="pk-spinner pk-spinner-success"></span>
<span class="pk-spinner pk-spinner-error"></span>

<!-- Inline inside text or button -->
Loading <span class="pk-spinner pk-spinner-sm pk-spinner-inline"></span>

<button class="pk-btn pk-btn-primary" disabled>
  Saving <span class="pk-spinner pk-spinner-sm pk-spinner-inline pk-spinner-white"></span>
</button>

API

ClassDescription
pk-spinnerBase spinner (md size, primary color)
pk-spinner-sm16px / 2px border
pk-spinner-md24px / 3px border (explicit md)
pk-spinner-lg36px / 4px border
pk-spinner-xl48px / 5px border
pk-spinner-inlinedisplay:inline-block; vertical-align:middle
pk-spinner-primaryBlue (default)
pk-spinner-secondaryGrey
pk-spinner-successGreen
pk-spinner-errorRed
pk-spinner-whiteWhite (for dark backgrounds)
