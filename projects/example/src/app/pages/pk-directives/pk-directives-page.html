pkCopyToClipboard

Copies the bound string to the clipboard on click. Emits (pkCopied) with true on success or false on failure.

{{ codeCopyToClipboard }}

pkAutoFocus

Focuses the host element after the view initialises. Set [pkAutoFocus]="false" to disable conditionally.

pkDebounceClick

Debounces rapid clicks and emits (pkDebounceClicked) only after the configured delay (default 300 ms).

pkNumberOnly

Restricts an <input> to numeric keys only. Blocks letters, symbols and paste of non-numeric text. Set [pkAllowDecimal]="true" for decimal input.
