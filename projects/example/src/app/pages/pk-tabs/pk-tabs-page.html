pk-tabs

Accessible tabbed navigation using content projection and Angular signals.

Demo

⚙️ Settings content — preferences, notifications.

💳 Billing content — plan, invoices.

Installation

import { PkTabs, PkTab } from 'ngx-pk-ui';

@Component({
  imports: [PkTabs, PkTab],
  // ...
})

Usage

<pk-tabs>
  <pk-tab label="Profile">
    <p>Profile content here</p>
  </pk-tab>
  <pk-tab label="Settings">
    <p>Settings content here</p>
  </pk-tab>
  <pk-tab label="Disabled" [disabled]="true">
    <p>Never shown</p>
  </pk-tab>
</pk-tabs>

API — PkTab inputs

InputTypeRequiredDefaultDescription
labelstring✅—Tab header label text
disabledboolean❌falsePrevents tab from being selected
