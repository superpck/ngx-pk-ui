pk-toastr

Non-blocking toast notifications. Inject the service anywhere — no module needed.

Example

Installation

<!-- Add once to your root component template -->
<pk-toastr />
import { PkToastr } from 'ngx-pk-ui';

@Component({
  imports: [PkToastr],
  template: `<router-outlet /><pk-toastr />`,
})

Usage

import { PkToastrService } from 'ngx-pk-ui';

@Component({ ... })
export class MyComponent {
  toastr = inject(PkToastrService);

  save() {
    this.toastr.success('Saved!', 'Success');
    this.toastr.error('Failed', 'Error', 0);      // duration=0 → persist
    this.toastr.warning('Check your input');
    this.toastr.info('Processing…', undefined, 8000);
    this.toastr.dismiss(id);                       // remove one
    this.toastr.clear();                           // remove all
  }
}

API

MethodParametersDescription
success(message, title?, duration?)Green toast
error(message, title?, duration?)Red toast
info(message, title?, duration?)Blue toast
warning(message, title?, duration?)Orange toast
dismiss(id: number)Remove one toast by ID
clear—Remove all toasts

Default duration is 4000ms. Pass 0 to keep until manually dismissed.
