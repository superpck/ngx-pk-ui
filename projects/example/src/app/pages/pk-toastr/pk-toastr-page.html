pk-toastr

Non-blocking toast notifications. Inject the service anywhere — no module needed.

Example

Position

Progress bar

Installation

npm install ngx-pk-ui

No template tag needed. PkToastrService automatically mounts the toast container into document.body the first time it is injected. Just inject the service and start calling it. 

import { PkToastrService } from 'ngx-pk-ui';

@Component({
  // No PkToastr in imports[], no <pk-toastr /> in template
  imports: [],
})
export class AppComponent {
  toastr = inject(PkToastrService); // ← this is all you need
}

Usage

import { PkToastrService } from 'ngx-pk-ui';

@Component({ ... })
export class MyComponent {
  toastr = inject(PkToastrService);

  save() {
    this.toastr.success('Saved!', 'Success');
    this.toastr.error('Failed', 'Error', { duration: 0 });                    // persist
    this.toastr.warning('Check your input');
    this.toastr.info('Processing…', undefined, { duration: 8000 });
    this.toastr.success('Bottom right!', undefined, { position: 'bottom-right' });
    this.toastr.warning('No progress', undefined, { progress: false });      // hide bar
    this.toastr.success('Banner!', undefined, { position: 'top-full' });      // full-width top
    this.toastr.error('Offline', undefined, { position: 'bottom-full', duration: 0 }); // sticky bottom
    this.toastr.dismiss(id);                                                   // remove one
    this.toastr.clear();                                                       // remove all
  }
}

API

MethodParametersDescription
success(message, title?, options?)Green toast
error(message, title?, options?)Red toast
info(message, title?, options?)Blue toast
warning(message, title?, options?)Orange toast
dismiss(id: number)Remove one toast by ID
clear—Remove all toasts
