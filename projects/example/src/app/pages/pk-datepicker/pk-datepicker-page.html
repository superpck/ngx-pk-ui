pk-datepicker

Date picker with locale support (TH/EN), min/max range, and clear action.

Limit min/max date

Selected: {{ dateValue() ? (dateValue() | date:'yyyy-MM-dd') : 'null' }}

Locale

EN: TH:

Usage

import { FormsModule } from '@angular/forms';
import { PkDatepickerComponent } from 'ngx-pk-ui';

@Component({
  imports: [FormsModule, PkDatepickerComponent],
})
export class AnyPage {
  dateValue: Date | null = null;
}

<pk-datepicker
  [(ngModel)]="dateValue"
  locale="TH"
  [minDate]="minDate"
  [maxDate]="maxDate"
  [setNull]="true"
/>

API

Input/Output Type Default Description
[(ngModel)] Date | null null Two-way selected date value
disabled boolean false Disable date picker interaction
setNull boolean true Show clear button to reset value
dateInput boolean true Allow typing date in input field
locale 'TH' | 'EN' 'TH' Display language and date formatting locale
startOfWeek 'monday' | 'sunday' 'monday' Calendar week start day
minDate Date | null null Minimum selectable date
maxDate Date | null null Maximum selectable date
placeholder string '' Input placeholder text
style string | null null Inline style string for input width/layout
(onDateChange) EventEmitter<Date | null> — Emits when selected date changes
