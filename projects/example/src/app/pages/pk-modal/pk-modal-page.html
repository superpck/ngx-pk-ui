pk-modal

Modal dialog with content projection slots, backdrop blur, size variants, and close control.

Basic (header / body / footer)

Modal Title

This is the modal body. You can put any content here — forms, tables, text.

Click the × button, the backdrop, or Cancel to close.

Theme variants

theme="{{ currentTheme() }}"

This modal uses theme="{{ currentTheme() }}".

Theme white and none keep the original style.

closeMarker=false, closeAny=false (no close button / no backdrop dismiss)

Non-dismissible Modal

The × button and backdrop click are disabled. You must use the button below to close.

blur=false (no backdrop blur)

No Blur Backdrop The backdrop has a dimmed overlay but no blur filter.

Size variants

size="{{ currentSize() }}" This modal uses size="{{ currentSize() }}".

customClass & customStyle

Custom Styled Modal This modal has customClass="my-themed-modal" and [customStyle]="{'border': '2px solid #43a047'}".

Installation

import { PkModalModule } from 'ngx-pk-ui';

@Component({
  imports: [PkModalModule],
  // ...
})

Usage

<!-- Template -->
<button (click)="showModal = true">Open</button>

<pk-modal
  [openModal]="showModal"
  [size]="'lg'"
  [blur]="true"
  [closeAny]="true"
  (onClose)="showModal = false"
>
  <pk-modal-header>Title</pk-modal-header>

  <pk-modal-body>
    <p>Any content here.</p>
  </pk-modal-body>

  <pk-modal-footer>
    <button class="pk-btn pk-btn-secondary pk-btn-outline" (click)="showModal = false">Cancel</button>
    <button class="pk-btn pk-btn-primary" (click)="confirm()">OK</button>
  </pk-modal-footer>
</pk-modal>

closeMarkerbooleantrueShow × button

customStyleRecord<string,string> | nullnullInline styles applied to the dialog container
