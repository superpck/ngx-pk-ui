Installation

import {{ '{' }} PkTimeline, PkTimelineItem {{ '}' }} from 'ngx-pk-ui';

@Component({{ '{' }}
  imports: [PkTimeline, PkTimelineItem],
{{ '}' }})

Usage

<pk-timeline direction="vertical" lineStyle="solid">
  <pk-timeline-item
    label="16 Oct"
    sublabel="09:15"
    icon="check_circle"
    [active]="true"
    dotColor="#10b981">
    <p>Content goes here</p>
  </pk-timeline-item>

  <!-- with avatar image -->
  <pk-timeline-item label="15 Oct" image="https://example.com/avatar.jpg">
    <p>Content goes here</p>
  </pk-timeline-item>
</pk-timeline>

API — pk-timeline (PkTimeline)

Inputs

InputTypeDefaultDescription
direction'vertical' | 'horizontal''vertical'Layout direction of the timeline
lineStyle'solid' | 'dashed''solid'Style of the connecting line between items

API — pk-timeline-item (PkTimelineItem)

Inputs

InputTypeDefaultDescription
labelstring''Primary label (date, time, step name) shown beside the dot
sublabelstring''Secondary label shown below the main label
iconstring''Material Symbols icon name — shown inside the dot (overridden by image)
imagestring''URL of an avatar/photo — displayed as a circular image in the dot (highest priority)
dotColorstring''CSS color override for the dot border and fill (e.g. '#10b981')
activebooleanfalseFilled dot with primary color background; icon turns white

Dot priority: image > icon > empty bordered circle

CSS Custom Properties

VariableDefaultDescription
--pk-timeline-colorvar(--pk-btn-primary, #1976d2)Global dot and line color — override per-item with dotColor input
