pk-toggle

Toggle switch replacing <input type="checkbox">. Pure CSS — no JavaScript. Fully accessible and keyboard-operable.

Basic

Color variants

Sizes

Label position

In a form

Installation

// angular.json
"styles": ["node_modules/ngx-pk-ui/styles/pk-toggle.css"]

// or include everything
"styles": ["node_modules/ngx-pk-ui/styles/pk-ui.css"]

// or CSS import
@import 'ngx-pk-ui/styles/pk-toggle.css';

Usage

<!-- Minimal -->
<label class="pk-toggle">
  <input type="checkbox" />
  <span class="pk-toggle__track"></span>
</label>

<!-- With label -->
<label class="pk-toggle pk-toggle-success">
  <input type="checkbox" checked />
  <span class="pk-toggle__track"></span>
  <span class="pk-toggle__label">Enable feature</span>
</label>

<!-- Small, label on left -->
<label class="pk-toggle pk-toggle-sm pk-toggle-label-left">
  <input type="checkbox" />
  <span class="pk-toggle__track"></span>
  <span class="pk-toggle__label">Compact</span>
</label>

<!-- Angular two-way binding -->
<label class="pk-toggle">
  <input type="checkbox" [(ngModel)]="isEnabled" />
  <span class="pk-toggle__track"></span>
  <span class="pk-toggle__label">Active</span>
</label>

Class reference

ClassApplies toDescription
pk-toggle<label>Wrapper — required
pk-toggle__track<span>The track + thumb — required
pk-toggle__label<span>Optional text label
pk-toggle-success<label>Green when checked
pk-toggle-warn<label>Orange when checked
pk-toggle-error<label>Red when checked
pk-toggle-sm<label>Small size (34×18px)
pk-toggle-lg<label>Large size (56×30px)
pk-toggle-label-left<label>Move label to the left
