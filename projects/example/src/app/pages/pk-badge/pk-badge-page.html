pk-badge

Status badges and pill indicators. Solid, outline, dot, and pill variants.

Solid variants

Primary Primary Success Warn Error

Outline

Primary Success Warn Error

Sizes

sm default lg

Pill

Default v2.2.0 Deprecated

Dot indicator

Inline with text / button

Notifications 9 Online

Usage

<!-- Solid variants -->
<span class="pk-badge">Primary</span>
<span class="pk-badge pk-badge-success">Success</span>
<span class="pk-badge pk-badge-warn">Warn</span>
<span class="pk-badge pk-badge-error">Error</span>

<!-- Outline -->
<span class="pk-badge pk-badge-error pk-badge-outline">Error</span>

<!-- Sizes -->
<span class="pk-badge pk-badge-sm">3</span>
<span class="pk-badge pk-badge-lg">New</span>

<!-- Pill -->
<span class="pk-badge pk-badge-success pk-badge-pill">v2.2.0</span>

<!-- Dot indicator -->
<span class="pk-badge pk-badge-dot pk-badge-success"></span>

<!-- Inline with button -->
<button class="pk-btn pk-btn-primary">
  Messages <span class="pk-badge pk-badge-error pk-badge-sm">3</span>
</button>

API

ClassDescription
pk-badgeBase badge (default = primary, solid)
pk-badge-primaryPrimary color (blue)
pk-badge-successSuccess color (green)
pk-badge-warnWarning color (orange)
pk-badge-errorError color (red)
pk-badge-outlineTransparent bg, colored border + text
pk-badge-smSmall size (16px height)
pk-badge-lgLarge size (28px height)
pk-badge-pillRectangular with rounded ends
pk-badge-dotEmpty circle indicator (10px)
