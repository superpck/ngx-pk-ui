pk-font

Thai & Lao Google Fonts helper classes. All fonts are served from Google Fonts CDN (SIL Open Font License 1.1).

Installation

pk-font.css is opt-in — not included in pk-ui.css by default. Import it separately when needed.

// angular.json
"styles": [
  "node_modules/ngx-pk-ui/styles/pk-ui.css",
  "node_modules/ngx-pk-ui/styles/pk-font.css"
]

// or CSS / SCSS
@import 'ngx-pk-ui/styles/pk-font.css';
