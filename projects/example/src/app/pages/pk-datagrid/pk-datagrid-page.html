pk-datagrid

Data grid with sorting, filtering, pagination, loading, and expandable row detail.

Demo

Usage

import { PkDatagridModule } from 'ngx-pk-ui';

@Component({
  imports: [PkDatagridModule],
})
export class AnyPage {}

API

pk-datagrid

InputTypeDefaultDescription
pkDgLoadingbooleanfalseShow loading overlay
itemsany[][]Row source data (usually set via *pkDgItems)
filterValuesRecord<string, string>{}Current column filter values
filterChangeEventEmitter<{ key, value }>—Emits when a filter input changes

pk-dg-column

InputTypeDescription
pkDgFieldstringField key for sorting
pkDgFilterKeystringField key used by filter popup
[style.width.px]numberColumn width in pixels

pk-dg-pagination

Input/OutputTypeDescription
pkDgPageSizenumberRows per page
totalItemsnumberTotal rows after filtering
pkDgPageChangeEventEmitter<number>Emits current page
pkDgPageSizeChangeEventEmitter<number>Emits selected page size

Directives

DirectiveUsageDescription
*pkDgItems*pkDgItems="let row of rows"Structural directive to render visible paged rows
*pkIfExpanded<pk-dg-row-detail *pkIfExpanded>...</pk-dg-row-detail>Renders row detail only when current row is expanded
