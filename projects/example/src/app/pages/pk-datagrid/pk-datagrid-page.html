Usage

import { PkDatagridModule } from 'ngx-pk-ui';

@Component({
  selector: 'app-my-page',
  imports: [PkDatagridModule],
  template: `
    <pk-datagrid [pkDgLoading]="loading">

      <!-- Column headers (sort / filter optional) -->
      <pk-dg-header [pkDgSort]="'name'" [pkDgFilter]="'name'" [style.width.px]="200">Name</pk-dg-header>
      <pk-dg-header [pkDgSort]="'role'" [style.width.px]="140">Role</pk-dg-header>

      <!-- Rows (structural directive handles paging) -->
      <pk-dg-rows *pkDgRows="let row of rows" [pkDgRow]="row">

        <!-- Optional action menu -->
        <pk-dg-action>
          <button class="action-item">Edit</button>
          <button class="action-item" style="color:red">Delete</button>
        </pk-dg-action>

        <!-- Data cells (one per header, in the same order) -->
        <pk-dg-column>{{ '{' }}{{ '{' }} row.name {{ '}' }}{{ '}' }}</pk-dg-column>
        <pk-dg-column>{{ '{' }}{{ '{' }} row.role {{ '}' }}{{ '}' }}</pk-dg-column>

        <!-- Optional expandable row detail -->
        <pk-dg-row-expand *pkDgRowIsExpand>
          <p>{{ '{' }}{{ '{' }} row.detail {{ '}' }}{{ '}' }}</p>
        </pk-dg-row-expand>
      </pk-dg-rows>

      <!-- Footer with pagination -->
      <pk-dg-footer>
        <pk-dg-pagination #p [pkDgPageSize]="10" [rowCount]="rows.length">
          <pk-dg-page-size [pkPageSizeList]="[10, 20, 50]">Rows per page</pk-dg-page-size>
          {{ '{' }}{{ '{' }} p.firstItem + 1 {{ '}' }}{{ '}' }} - {{ '{' }}{{ '{' }} p.lastItem + 1 {{ '}' }}{{ '}' }} of {{ '{' }}{{ '{' }} rows.length {{ '}' }}{{ '}' }} rows
        </pk-dg-pagination>
      </pk-dg-footer>
    </pk-datagrid>
  `,
})
export class AnyPage {
  loading = false;
  rows = [
    { name: 'Alice', role: 'Admin',     detail: 'alice@example.com' },
    { name: 'Bob',   role: 'Developer', detail: 'bob@example.com'   },
  ];
}

Data source

Demo data is generated by Random User Generator — a free, open API for generating random user data.

API

pk-datagrid

Input Type Default Description
pkDgLoading boolean false Show loading overlay

pk-dg-rows

Input Type Default Description
pkDgRow any — The data object for this row (used by sort/filter directives)
rowClass string '' CSS class added to the <tr> element — useful for conditional row highlighting, e.g. [rowClass]="row.active ? 'row-active' : ''"

pk-dg-header

Input Type Description
pkDgSort string Field key for sorting
pkDgFilter string Field key used by filter popup
[style.width.px] number Column width in pixels (default 120)

pk-dg-pagination

Input / Property Type Description
pkDgPageSize number Rows per page (default: 10)
rowCount number Total rows (set automatically via *pkDgRows)
firstItem number 0-based index of the first row on the current page
lastItem number 0-based index of the last row on the current page
pkDgPageChange EventEmitter<number> Emits current page number
pkDgPageSizeChange EventEmitter<number> Emits selected page size

Directives

Directive Usage Description
*pkDgRows *pkDgRows="let row of rows" Structural directive to render visible paged rows
*pkDgRowIsExpand <pk-dg-row-expand *pkDgRowIsExpand>...</pk-dg-row-expand> Renders row detail only when current row is expanded
