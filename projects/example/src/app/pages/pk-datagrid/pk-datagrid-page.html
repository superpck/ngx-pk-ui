pk-datagrid

Data grid with sorting, filtering, pagination, loading, and expandable row detail.

Usage

import { PkDatagridModule } from 'ngx-pk-ui';

@Component({
  imports: [PkDatagridModule],
})
export class AnyPage {}

API

pk-datagrid

InputTypeDefaultDescription
pkDgLoadingbooleanfalseShow loading overlay

pk-dg-header

InputTypeDescription
pkDgSortstringField key for sorting
pkDgFilterstringField key used by filter popup
[style.width.px]numberColumn width in pixels

pk-dg-pagination

Input / PropertyTypeDescription
pkDgPageSizenumberRows per page (default: 10)
rowCountnumberTotal rows (set automatically via *pkDgRows)
firstItemnumber0-based index of the first row on the current page
lastItemnumber0-based index of the last row on the current page
pkDgPageChangeEventEmitter<number>Emits current page number
pkDgPageSizeChangeEventEmitter<number>Emits selected page size

Directives

DirectiveUsageDescription
*pkDgRows*pkDgRows="let row of rows"Structural directive to render visible paged rows
*pkDgRowIsExpand<pk-dg-row-expand *pkDgRowIsExpand>...</pk-dg-row-expand>Renders row detail only when current row is expanded
