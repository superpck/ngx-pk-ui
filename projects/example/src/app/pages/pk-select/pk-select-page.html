pk-select

Single and multi select dropdown with search support and ngModel integration.

Interactive demo

@if (mode() === 'single') {
Selected: {{ selectedSingle() ?? 'None' }}
} @else {
Selected ({{ selectedMulti().length }}): {{ selectedMulti().join(', ') || 'None' }}
}

With ngModel

ngModel value: {{ selectedNgModel ?? 'null' }}

Usage

import { PkSelectComponent, type SelectOption } from 'ngx-pk-ui';
import { FormsModule } from '@angular/forms';

@Component({
  imports: [FormsModule, PkSelectComponent],
})
export class AnyPage {
  options: SelectOption[] = [
    { label: 'Bangkok', value: 'bangkok' },
    { label: 'Chiang Mai', value: 'chiangmai' },
  ];

  selected: string | null = null;
}

<pk-select
  [options]="options"
  mode="single"
  [searchable]="true"
  placeholder="Select"
  [(ngModel)]="selected"
/>

API

Input/OutputTypeDefaultDescription
optionsSelectOption[]requiredOptions list for dropdown
mode'single' | 'multi''single'Select mode behavior
placeholderstring'เลือก...'Placeholder when no value selected
searchablebooleanfalseEnable search input in dropdown
disabledbooleanfalseDisable interaction
labelFieldstring'label'Field used as label text
valueFieldstring'value'Field used as value key
returnObjectsbooleanfalseEmit selected object(s) instead of primitive values
(change)EventEmitter<any>—Emits selected value(s) on change
[(ngModel)]any—Two-way binding through ControlValueAccessor

SelectOption: label?, value?, disabled?, and custom fields.
