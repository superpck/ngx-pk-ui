pk-card

Card layout with header, body, and footer sections. Color variants and modifiers included.

Basic structure

Card Title
Some content inside the card body. It can be anything — text, form, media.
Body only — no header or footer.
No Footer
Just a header and body.

Color variants

Primary
pk-card-primary
Secondary
pk-card-secondary
Success
pk-card-success
Error
pk-card-error

Modifiers

pk-card-shadow
Elevated shadow.
pk-card-flat
No shadow.
pk-card-outlined
Border only, no shadow.

Usage

<!-- Full structure -->
<div class="pk-card">
  <div class="pk-card-header">Title</div>
  <div class="pk-card-body">Content goes here.</div>
  <div class="pk-card-footer">
    <button class="pk-btn pk-btn-primary pk-btn-sm">OK</button>
  </div>
</div>

<!-- Color variants -->
<div class="pk-card pk-card-primary">...</div>
<div class="pk-card pk-card-success">...</div>
<div class="pk-card pk-card-error">...</div>

<!-- Modifiers -->
<div class="pk-card pk-card-shadow">...</div>
<div class="pk-card pk-card-flat">...</div>
<div class="pk-card pk-card-outlined">...</div>

API

ClassElementDescription
pk-cardContainerBase card (white bg, border, radius, shadow)
pk-card-headerChildTop section — bold title, bottom border
pk-card-bodyChildMain content area — flex:1
pk-card-footerChildBottom section — light bg, flex row
pk-card-primaryContainerBlue header bg + matching border
pk-card-secondaryContainerGrey header bg + matching border
pk-card-successContainerGreen header bg + matching border
pk-card-errorContainerRed header bg + matching border
pk-card-shadowContainerElevated box shadow
pk-card-flatContainerNo shadow
pk-card-outlinedContainerBorder only, no shadow
