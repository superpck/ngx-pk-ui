Usage

<!-- angular.json -->
"styles": ["node_modules/ngx-pk-ui/styles/pk-ui.css"]

<!-- Fixed columns -->
<div class="pk-row">
  <div class="pk-col-6">50%</div>
  <div class="pk-col-6">50%</div>
</div>

<!-- Responsive -->
<div class="pk-row">
  <div class="pk-col-12 pk-col-sm-6 pk-col-md-4">...</div>
</div>

<!-- Auto width -->
<div class="pk-row">
  <div class="pk-col">fills remaining space</div>
</div>

Breakpoints

Prefix Min-width Classes
(none) Always pk-col-[1-12]
xs Always (<576px) pk-col-xs-[1-12]
sm ≥ 576px pk-col-sm-[1-12]
md ≥ 768px pk-col-md-[1-12]
lg ≥ 992px pk-col-lg-[1-12]
xl ≥ 1200px pk-col-xl-[1-12]

Row modifiers

Class Description
pk-row--gap-0 Remove gap between columns
pk-row--gap-sm Small gap (8px)
pk-row--gap-lg Large gap (32px)
pk-row--align-center Vertically center columns
pk-row--align-top Align columns to top
pk-row--align-bottom Align columns to bottom
pk-row--justify-between Space between columns
pk-row--justify-center Center columns horizontally
pk-row--justify-end Right-align columns
pk-row--no-wrap Prevent column wrapping
pk-row--reverse Reverse column order
