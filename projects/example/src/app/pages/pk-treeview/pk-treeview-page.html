pk-treeview

Hierarchical tree navigation with expand/collapse and selection modes.

Demo

Selected: {{ selectedCount() }}

Usage

import { PkTreeviewModule, type TreeNode } from 'ngx-pk-ui';

@Component({
  imports: [PkTreeviewModule],
})
export class AnyPage {
  data: TreeNode[] = [
    { label: 'Root', children: [{ label: 'Child' }] }
  ];
}

<pk-treeview
  [nodes]="data"
  selection="multi"
  [showSelectAll]="true"
  [expandDepth]="1"
  (selected)="onSelected($event)"
/>

API

Input/OutputTypeDefaultDescription
nodesTreeNode[][]Tree data source
selection'none' | 'single' | 'multi''none'Selection behavior
showSelectAllbooleantrueShow select-all row in multi mode
expandDepthnumber1Initial expanded depth
(selected)EventEmitter<TreeNode[]>—Emits current selected nodes
(nodeToggle)EventEmitter<TreeNode>—Emits when a node is expanded/collapsed

TreeNode fields: label, icon?, routerLink?, data?, children?.
