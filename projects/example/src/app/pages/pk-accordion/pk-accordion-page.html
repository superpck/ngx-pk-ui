pk-accordion

Collapsible panels using content projection and Angular signals. Supports single and multi-open modes.

Single mode (default)

Only one panel can be open at a time.

A lightweight Angular 21 component library published to npm — batteries included, zero dependencies beyond Angular itself.

Run npm install ngx-pk-ui then import the components you need in your standalone component's imports[] array.

Installation

import { PkAccordion, PkAccordionItem } from 'ngx-pk-ui';

@Component({
  imports: [PkAccordion, PkAccordionItem],
  // ...
})

Usage

<pk-accordion>
  <pk-accordion-item label="Section 1">
    <p>Content goes here.</p>
  </pk-accordion-item>
  <pk-accordion-item label="Section 2" [open]="true">
    <p>Starts expanded.</p>
  </pk-accordion-item>
  <pk-accordion-item label="Disabled" [disabled]="true">
    <p>Cannot be opened.</p>
  </pk-accordion-item>
</pk-accordion>

<!-- Multi mode -->
<pk-accordion [multi]="true">
  <pk-accordion-item label="A">...</pk-accordion-item>
  <pk-accordion-item label="B">...</pk-accordion-item>
</pk-accordion>
