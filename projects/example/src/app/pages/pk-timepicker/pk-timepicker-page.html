API

Input / Output Type Default Description
format 'hms' | 'hm' | 'h' 'hm' Columns shown: hours+minutes+seconds / hours+minutes / hours only
type '24H' | '12H' '24H' Clock mode — 12H shows AM/PM toggle
inputType 'spinner' | 'number' | 'dropdown' 'number' UI style — spinner (arrows), number (text input, reverts if invalid), dropdown (select list)
customClass string '' Extra CSS class on wrapper
customStyle Record<string, string> {{ '{' }}{{ '}' }} Inline styles on wrapper
(onTimeChange) string — Emits 24H value string on every change
ngModel string '' Two-way binding — value format: "HH:mm:ss" / "HH:mm" / "HH"

Value format

format type ngModel value example
hm 24H "14:30"
hms 24H "14:30:45"
h 24H "14"
hm 12H "14:30" (displayed as 02:30 PM)

The ngModel value is always 24H regardless of the type setting. type="12H" only affects the display.
