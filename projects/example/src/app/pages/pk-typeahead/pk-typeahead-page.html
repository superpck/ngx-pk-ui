pk-typeahead

Input suggestion dropdown for fast text matching with keyboard navigation and custom-value control.

Selected value: {{ selectedValue || 'None' }}

String items (strict)

Selected value: {{ selectedStrict || 'None' }}

Usage

import { FormsModule } from '@angular/forms';
import { PkTypeaheadComponent } from 'ngx-pk-ui';

@Component({
  imports: [FormsModule, PkTypeaheadComponent],
})
export class AnyPage {
  items = [
    { label: 'Bangkok', value: 'bangkok' },
    { label: 'Phuket', value: 'phuket' },
  ];

  selected = '';
}

<pk-typeahead
  [items]="items"
  labelField="label"
  valueField="value"
  [(ngModel)]="selected"
/>

API

Input/OutputTypeDefaultDescription

valueFieldstring'value'Bound value field for object items
allowCustomValuebooleantrueAllow values outside suggestion list

(itemSelected)EventEmitter<any>—Emits selected suggestion item
[(ngModel)]any—Two-way value binding
