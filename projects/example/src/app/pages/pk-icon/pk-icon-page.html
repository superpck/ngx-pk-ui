Material Symbols mode (local-first)

Use iconSet="material-symbols" to render Google Material Symbols through local font files in styles/fonts/material-symbols.

{{ materialName() }} 
<pk-icon
  name="{{ materialName() }}"
  iconSet="material-symbols"
  variant="{{ materialVariant() }}"
  [weight]="{{ materialWeight() }}"
  [fill]="{{ materialFill() }}"
  [opticalSize]="48"
  [size]="48"
/>

Installation

npm install ngx-pk-ui
import { PkIcon } from 'ngx-pk-ui';
import type { PkIconName } from 'ngx-pk-ui';

@Component({
  imports: [PkIcon],
  template: `<pk-icon name="search" />`,
})
