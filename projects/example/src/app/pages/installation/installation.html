Installation

Get started with ngx-pk-ui in your Angular {{angularVersion.major}} project in four steps.

1. Install

npm install ngx-pk-ui

2. Add global styles

Add one stylesheet entry to your angular.json to load all components' styles and CSS utilities at once. 

"styles": [
  "node_modules/ngx-pk-ui/styles/pk-ui.css"
]

Or import in your global CSS file:

@import 'ngx-pk-ui/styles/pk-ui.css';

You can also import individual modules instead of the full bundle: 

@import 'ngx-pk-ui/styles/pk-grid.css';
@import 'ngx-pk-ui/styles/pk-btn.css';
@import 'ngx-pk-ui/styles/pk-table.css';
Note: pk-font.css (Thai/Lao Google Fonts) is not included in pk-ui.css — import it separately if needed: @import 'ngx-pk-ui/styles/pk-font.css';

3. Import components

Every component is standalone — add only what you need to each component's imports[] array. 

import { PkToastr, PkAlert, PkModal, PkTooltip } from 'ngx-pk-ui';

@Component({
  imports: [PkToastr, PkAlert, PkModal, PkTooltip],
  template: `
    <pk-toastr />
    <pk-alert />
    <router-outlet />
  `,
})
export class AppComponent { }
Tip: pk-toastr and pk-alert self-mount into <body> automatically when their service is first injected — no template tag required. Add them to AppComponent once, or skip the tag entirely.

4. Inject and use services

import { PkToastrService, PkAlertService } from 'ngx-pk-ui';

@Component({
  imports: [],   // no component import needed for service-only use
})
export class MyComponent {
  toastr = inject(PkToastrService);
  alert  = inject(PkAlertService);

  save() {
    this.toastr.success('Saved!');
  }

  async deleteItem() {
    const ok = await this.alert.confirm('Delete this item?');
    if (ok) { /* proceed */ }
  }
}

NgModule-based components

A few components use NgModule pattern for internal complexity. Import their module instead of the component directly.

Module Import
PkTabsModule import { PkTabsModule } from 'ngx-pk-ui'
PkDatagridModule import { PkDatagridModule } from 'ngx-pk-ui'
PkTreeviewModule import { PkTreeviewModule } from 'ngx-pk-ui'
PkModalModule import { PkModalModule } from 'ngx-pk-ui' (or standalone slots)
