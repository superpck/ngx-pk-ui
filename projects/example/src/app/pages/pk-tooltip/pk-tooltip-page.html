pk-tooltip

Tooltip directive — use pkTooltip on any element as a replacement for the native [title] attribute. Supports four positions and four color types.

Positions

Types

On any element

Hover this text v1.1.2

Usage

import { PkTooltip } from 'ngx-pk-ui';

@Component({
  imports: [PkTooltip],
  // ...
})
export class MyComponent {}

<!-- Positions: top (default) | bottom | left | right -->
<button pkTooltip="Saved!" pkTooltipPosition="top">Save</button>
<button pkTooltip="Deleted" pkTooltipPosition="bottom">Delete</button>

<!-- Types: primary (default) | success | danger | info -->
<button pkTooltip="All good" pkTooltipType="success">OK</button>
<button pkTooltip="Warning!" pkTooltipType="danger">Delete</button>

<!-- Combined -->
<button
  pkTooltip="Are you sure?"
  pkTooltipPosition="right"
  pkTooltipType="danger">
  Remove
</button>

API

Input Type Default Description
pkTooltip string required Text to display in the tooltip
pkTooltipPosition 'top' | 'bottom' | 'left' | 'right' 'top' Where the tooltip appears relative to the host element
pkTooltipType 'primary' | 'success' | 'danger' | 'info' 'primary' Color variant of the tooltip bubble
