pk-qrcode

QR Code generator — versions 1–40, all error correction levels (L/M/Q/H), center logo support. Pure TypeScript Reed-Solomon encoder. No external dependencies.

PkQrcode Inputs

Input Type Default Description
value string required Text / URL to encode
ecLevel PkQrEcLevel 'M' Error correction: L (7%) / M (15%) / Q (25%) / H (30%)
size number 200 Rendered pixel size (width = height)
darkColor string '#000000' Dark module color
lightColor string '#ffffff' Light module / background color
logo string | null null Center logo image URL; auto-upgrades ECL to Q
logoSize number 0.25 Logo size as fraction of QR (0.1–0.3)
margin number 4 Quiet zone width in modules

PkQrcode Methods

Method Description
downloadSvg() Download QR code as .svg file
downloadPng() Download QR code as .png file
