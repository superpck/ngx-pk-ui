pk-layout

A pure-CSS layout shell: fixed top navbar + fixed sidebar. All dimensions and colors are CSS variables — override in :root or any parent selector.

Usage

Import from pk-ui.css (already included) or directly:

<div class="pk-layout">

  <!-- Fixed top navbar -->
  <nav class="pk-navbar">
    <button class="pk-navbar__hamburger">...</button>
    <div class="pk-navbar__brand">My App</div>
    <div class="pk-navbar__content">
      <span class="pk-navbar__title">Page Title</span>
    </div>
    <div class="pk-navbar__end">
      <!-- avatar, buttons, etc. -->
    </div>
  </nav>

  <!-- Mobile backdrop (show/hide via JS) -->
  <button class="pk-layout__backdrop pk-layout__backdrop--open"
          (click)="closeSidebar()"></button>

  <!-- Fixed sidebar -->
  <nav class="pk-sidebar pk-sidebar--open">
    <div class="pk-sidebar__section">
      <span class="pk-sidebar__heading">General</span>
      <a class="pk-sidebar__link active" href="#">Home</a>
      <a class="pk-sidebar__link" href="#">Settings</a>
    </div>
    <div class="pk-sidebar__divider"></div>
    <div class="pk-sidebar__section">
      <span class="pk-sidebar__heading">Reports</span>
      <a class="pk-sidebar__link" href="#">Sales</a>
    </div>
  </nav>

  <!-- Main content -->
  <main class="pk-layout__main">
    <router-outlet />
  </main>

</div>

Responsive behaviour

ViewportNavbarSidebarMain
> 768px Brand slot fixed width = sidebar width, right border. Hamburger hidden. Always visible, fixed left starting below navbar. margin-left: var(--pk-sidebar-width)
≤ 768px Brand has no fixed width or right border. Hamburger shown inside navbar. Hidden (translateX(-110%)) by default; slides in on --open with backdrop overlay. margin-left: 0, reduced padding.

Override example

Set CSS variables on any ancestor to theme the layout:

:root {{ '{' }}
  --pk-navbar-bg: #1e293b;
  --pk-navbar-text: #f1f5f9;
  --pk-navbar-border: #334155;
  --pk-sidebar-bg: #0f172a;
  --pk-sidebar-link-color: #94a3b8;
  --pk-sidebar-link-hover-bg: rgba(255, 255, 255, 0.06);
  --pk-sidebar-link-hover-color: #f1f5f9;
  --pk-sidebar-link-active-color: #38bdf8;
  --pk-sidebar-link-active-bg: rgba(56, 189, 248, 0.1);
  --pk-sidebar-link-active-border: #38bdf8;
  --pk-sidebar-heading-color: #475569;
  --pk-sidebar-border: #1e293b;
  --pk-layout-bg: #111827;
{{ '}' }}
