pk-btn

Button CSS utilities — variants, outline, shadow, group, and sizes.

Variants (solid)

@for (btn of buttonList; track btn.name) { }

Outline

@for (btn of buttonList; track btn.name) { }

Sizes

Shadow

Button group

Usage

<!-- Solid variants -->
<button class="pk-btn">Default (primary)</button>
<button class="pk-btn pk-btn-success">Success</button>
<button class="pk-btn pk-btn-error">Error</button>
<button class="pk-btn pk-btn-secondary">Secondary</button>

<!-- Outline modifier -->
<button class="pk-btn pk-btn-primary pk-btn-outline">Outline</button>

<!-- Shadow modifier -->
<button class="pk-btn pk-btn-primary pk-btn-shadow">Shadow</button>

<!-- Sizes -->
<button class="pk-btn pk-btn-sm">Small</button>
<button class="pk-btn pk-btn-lg">Large</button>

<!-- Group -->
<div class="pk-btn-group">
  <button class="pk-btn pk-btn-secondary">A</button>
  <button class="pk-btn pk-btn-secondary">B</button>
</div>

CSS Custom Properties

:root {
  --pk-btn-primary:    #1976d2;
  --pk-btn-secondary:  #546e7a;
  --pk-btn-success:    #43a047;
  --pk-btn-error:      #e53935;
}

Override any of these in your global :root to theme the entire library.

API Reference

Base Class

Class Description
pk-btn Base button class — required on all buttons

Variant Classes

Variant Name @for (btn of buttonList; track btn.name) { {{ btn.type }} }
Class Description
pk-btn-primary Primary variant — blue (#1976d2)
pk-btn-secondary Secondary variant — gray (#546e7a)
pk-btn-success Success variant — green (#43a047)
pk-btn-warn Warning variant — orange (#fb8c00)
pk-btn-error Error variant — red (#e53935)
pk-btn-... Other variant — which one of variant name

Modifier Classes

Combine with any variant

Class Description
pk-btn-outline Transparent background with colored border and text
pk-btn-shadow Adds box-shadow elevation
pk-btn-sm Small size — reduced padding and font size
pk-btn-lg Large size — increased padding and font size

Button Group

Class Description
pk-btn-group Wrapper <div> — groups adjacent buttons with collapsed borders

HTML Attribute

Attribute Description
disabled Native HTML attribute — reduces opacity and disables pointer events

Installation

// In angular.json styles array

// Or in global CSS/SCSS
@import 'ngx-pk-ui/styles/pk-btn.css';
