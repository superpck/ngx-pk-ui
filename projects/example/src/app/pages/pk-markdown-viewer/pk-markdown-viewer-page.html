pk-markdown-viewer

Render Markdown from a URL (fileName) or a raw string (content). Includes Print and Export actions.

Usage

Load a .md file by URL 

import { PkMarkdownViewer } from 'ngx-pk-ui';

@Component({
  imports: [PkMarkdownViewer],
  template: `
    <pk-markdown-viewer
      fileName="assets/CHANGELOG.md"
      theme="light"
      [showToolbar]="true"
    />
  `,
})

Render a raw Markdown string 

import { PkMarkdownViewer } from 'ngx-pk-ui';

@Component({
  imports: [PkMarkdownViewer],
  template: `
    <pk-markdown-viewer
      [content]="markdownString"
      theme="dark"
      title="My Document"
    />
  `,
})

Inputs

Input Type Default Description
fileName string '' URL to a .md file — fetched via fetch()
content string '' Raw Markdown string (takes priority over fileName)
theme 'light' | 'dark' 'light' Color theme
showToolbar boolean true Show/hide the toolbar (Print + Export buttons)
title string '' Override toolbar title (defaults to filename)
