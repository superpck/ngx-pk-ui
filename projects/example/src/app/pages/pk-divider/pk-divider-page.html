API

ClassDescription
pk-dividerBase horizontal divider (required)
pk-divider--verticalVertical inline separator (use on <span>)
pk-divider--leftLabel flush-left, long line on right
pk-divider--rightLabel flush-right, long line on left
pk-divider--dashedDashed line style
pk-divider--dottedDotted line style
pk-divider--md2 px line thickness
pk-divider--thick3 px line thickness
pk-divider--primaryPrimary color (blue)
pk-divider--secondarySecondary color (grey)
pk-divider--successSuccess color (green)
pk-divider--errorError color (red)
