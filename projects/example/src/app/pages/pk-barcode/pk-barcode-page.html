Method Description

Format notes

Format Valid input Checksum
code128 Printable ASCII 32–126 Auto-computed (mod 103)
code39 A–Z, 0–9, - . space $ / + % (auto-uppercase) None (optional)
ean13 12 or 13 digits Auto-computed if 12 digits given
ean8 7 or 8 digits Auto-computed if 7 digits given
