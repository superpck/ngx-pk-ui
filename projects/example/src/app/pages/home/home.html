ngx-pk-ui

An Angular {{angularVersion.major}} component library built for modern applications. It provides ready-to-use UI components and CSS utilities designed around Angular's signal-based architecture — fully standalone, no NgModules required.

Why ngx-pk-ui?

⚡
Signal-based
Built on Angular 22 signals — input(), contentChildren(), computed(). No @Input/@Output decorators.
🎨
Themeable
All colours driven by CSS custom properties. Override --pk-btn-primary and friends in :root to match your brand.
📦
Standalone
Every component is standalone — no NgModules required. Add only what you need to each component's imports[].
🌍
17 Locales
Thai, Lao, Arabic (RTL), Japanese, Korean, Chinese and 11 more. Calendar, datepicker, heatmap and pipes all share one locale token.
🔒
XSS-safe
HTML output is sanitized before rendering. pkHighlight escapes user input; the Markdown viewer escapes all raw HTML.
🧪
Well-tested
358 unit tests written with Vitest (Angular 22 default). Every component has a co-located .spec.ts.
