pkTruncate

Shortens a string to a maximum number of characters and appends an ellipsis.

{{ truncateText() | pkTruncate: truncateLimit() }}
Original length: {{ truncateText().length }} chars
{{ codeTruncate }}

pkTimeAgo

Renders a past date as a human-readable relative string. Uses pure: false so it updates automatically each change-detection cycle.

@for (p of timePresets; track p.ms) { }
{{ timeAgoDate() | pkTimeAgo }}

pkNumber

Formats a number with optional decimal places and thousand separators.

{{ numberValue() | pkNumber: numberDecimals() }} 
{{ codeNumber }}

pkFileSize

Formats a raw byte count to a human-readable string with unit (B, KB, MB, GB, TB).

{{ fileSizeBytes() | pkFileSize: fileSizeDecimals() }} 
{{ codeFileSize }}

pkHighlight

Wraps all occurrences of a search term inside <mark class="pk-highlight">. Use with [innerHTML]. The value is HTML-escaped first to prevent XSS.

pkDate

Formats a Date, timestamp, or ISO string using locale-aware month names. Supports Buddhist Era (BE) year output.

{{ demoDate | pkDate: dateFormat() : dateStyle() : dateLocale() : dateEra() }}
Input: new Date(2025, 0, 31) — January 31, 2025 CE

Format token reference

Token Output Example
d Day, no padding 1, 31
dd Day, zero-padded 01, 31
m Month — numeric or locale name (by style) 1, Jan, January
mm Month padded (numeric) or locale name 01, Jan, January
yyyy 4-digit year 2025, 2568
yy 2-digit year 25, 68
